"Бета-Банк" выяснил, что многие клиенты банка расторгают договор. Менеджеры банка обратили внимание, 
что клиенты покидают банк и и пришли к выводу, что дешевле удерживать оставшихся клиентов, чем привлекать новых. 

В нашем распоряжении информация о поведении клиентов, за некоторое время, кто-то из них уже ушел из банка (датасет Churn.csv).

Цель работы - построить модель для задачи классификации, которая спрогнозирует уйдет ли клиент из банка в ближайшее время.

Информацию о поведении клиентов, которые ушли или остались в банке, будем использовать для обучения 
и проверки качества модели.
 
Для поставленых целей подходят модели Decision Tree Classifier, Random Forest. Метрики качества - F1-мера и 
recall. Для проверки на тестовых данных выбрана обученная модель Random Forest, метрики качества на тестовых данных:
   F1-мера - 0,599;
   Recall - 0,603.

Прогноз модели будет передан менеджерам банка для дальнейшей работы.
